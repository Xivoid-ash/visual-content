Learning Efficient Convolutional Networks through Network Slimming(edit)
CNN 模型轻量化、通道剪枝、模型压缩
基于 BN 层缩放因子 + L1 正则化实现自动通道稀疏与剪枝，训练开销小、兼容现有架构、无需专用加速器
研究动机:
深度 CNN 性能强但体积大、内存高、计算量大，难以部署在移动端、IoT 等资源受限设备。
现有方法缺陷
低秩近似、权重量化、权重剪枝等大多只解决部分问题；
非结构化稀疏需要专用加速器才能提速；
部分结构化剪枝优化复杂、精度损失大。
研究目标
用简单方式同时实现：模型更小 + 内存更低 + 计算更少 + 精度不下降。
核心方法：Network Slimming
1. 核心思想
直接对BN 层的缩放因子 γ施加 L1 正则化，让不重要通道的 γ 自动趋近于 0，训练后剪掉 γ 小的通道，再微调恢复精度。
关键步骤
复用 BN 层 γ，不增加额外层；
L1 正则化，迫使模型自动筛选通道，实现通道级稀疏；
全局阈值剪枝：按比例统一剪掉低 γ 通道；
3. 标准流程（三步）
稀疏训练：正常训练 + BN 层 γ 加 L1 正则；
通道剪枝：全局阈值删除 γ 小的通道，得到窄模型；
微调恢复：小学习率微调，弥补剪枝带来的精度下降。
4. 进阶：多轮瘦身
重复 “稀疏训练→剪枝→微调”，可达到更高压缩率（VGG 可达 20 倍）。
5. 特殊结构适配
对 ResNet/DenseNet 等带残差 / 密集连接的网络：
在输入端做稀疏；
用通道选择层屏蔽无效通道，保证维度匹配。
实验结论
优点与创新
极简：只加 L1 正则，不改动网络结构；
通用：适配 VGG/ResNet/DenseNet 等主流架构；
实用：剪后模型无稀疏格式，常规硬件直接提速；
效果强：压缩率高、精度保留好，还带正则化收益。
复现关键
必须对所有 BN 层 γ加 L1；
用全局阈值而非分层剪枝；
剪后必须小学习率微调。
